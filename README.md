# Mario Bros game

You can find report in [`docs`](docs/index.md) folder.

## Setup project

```bash
python -m venv .venv
source .venv/bin/activate # for Linux
pip install -r requirements.txt
```

## Run

```bash
python -m game.main
```

## Build report

```bash
pip install -r dev_requirements.txt
mkdocs build
```
